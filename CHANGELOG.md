# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-12-01

### Added
- Multi-platform video download (TikTok, Instagram Reels, YouTube Shorts, Facebook Watch)
- TikTok watermark-free downloads via yt-dlp
- Video reframing to 9:16 vertical format
- Blurred and solid color background options
- Anti-copyright effects (mirror, speed adjustment, color tone)
- AI subtitle generation with OpenAI Whisper
- Multi-language transcription support
- Customizable subtitle styling (font, size, color, stroke)
- Automated TikTok upload via Playwright
- Cookie-based session persistence
- Viral content score calculation
- Trending hashtag detection and recommendation
- AI-powered description generation (OpenAI GPT / Ollama)
- Job queue system with parallel processing (3 workers)
- Analytics dashboard with statistics tracking
- Role-based authentication (admin, user, viewer)
- Automatic database backups (7-day retention)
- Modern Next.js 16 + React 19 frontend with dark mode
- Legacy Jinja2 dashboard template
- Docker Compose setup (backend, frontend, Redis, backup)
- RESTful API with 11 endpoints
- Environment-based configuration
- Windows batch scripts for easy startup
- Cookie export utilities for TikTok
- Comprehensive documentation
